# ResizeWizard

[https://resizewizard.app/](https://resizewizard.app/){: target="_blank" }

## Quick, anchored window resizing with themes and sync.

An efficient browser extension to quickly resize windows anchored to the top-left corner. Features include customizable themes, data synchronization, and seamless upgrades for enhanced productivity.
